Virtual Cluster States
  • New

    The virtual cluster definition is being validated.

  • Initializing

    A headnode is being created and configured for the virtual cluster.

  • Pending

    Compute nodes are being requested.

  • Running

    Compute nodes are running.

  • Terminating

    A terminate action has been received, and compute workers are being vacated.

  • Cleanup

    The headnode is shutdown, and other minor bookeping of the virtual cluster is deleted.

  • Terminated

    The cluster lifetime has ended and remains in this stage until it is explicitly deleted or relaunched.
